# HnuSec 2025 Win. ❄️冬季培训

## ❄️ 2025 冬季培训

本次招新主要面向海南大学24届全体新生，24届以外的学生也可以参与培训课程，并且在培训之后几天内，并提供基础考核学习成果，优异者通过考核同样可以进入HnuSec-Star队。本次除了CTF相关的WEB、PWN、REV、CRY、MISC五个方向之外，我们这次还计划开展开发相关的DEV课程。欢迎感兴趣的同学学习。

**考核与培训内容并非强关联，更多考验课后学习补充能力及知识拓展能力。**

[Q&A](./others/Q&A.md) 这个页面也许可以回答你的问题。
