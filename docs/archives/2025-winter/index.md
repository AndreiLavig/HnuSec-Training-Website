# HnuSec 2025 Winter ❄️ 冬季培训归档

## 📚 培训概述

本次冬季培训于2025年1月13日-2025年1月26日举行，为期14天。培训面向海南大学24级、25级全体学生，涵盖了CTF相关的WEB、PWN、REV、CRY、MISC五个方向。
